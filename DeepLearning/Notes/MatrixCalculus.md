​	**矩阵求导的本质：**

​	$\frac {dA}{dB}$：矩阵A中的每一个元素对矩阵B中的每一个元素求导。

​	我们从求导后的元素个数角度来看：
$$
A \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \  \ \ \ \  \ \ \  \  \ \ \ B →\ \frac {dA}{dB} \\
1\times1 \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ 1\times1 →1\times1\\
1\times p \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ 1\times n →p\times n \\
q\times p \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ m\times n→p\times q\times m\times n
$$
